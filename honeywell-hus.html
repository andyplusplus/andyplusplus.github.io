Honeywell Universal Surveillance (HUS) --The Smart Platform for IP Security Integration--New Edition 3.1 Release! A most reliable and cost effective IP solution

Dec 24, 2010
Page Content
Honeywell Security Group now announce to release a new generation of IP Video
Surveillance Solution with a comprehensive range of end-to-end offers. The new edition 3.1 of
Honeywell Universal Surveillance (HUS) is designed with various modules to accommodate various
customer needs, as well as applications down to the field.

The updated HUS 3.1 comprises HUS XPRO, HUS XACT and a new Pioneer series of IP cameras. This new edition represents a true end-to-end IP security solution-from the front end with full range of IP cameras covering fixed dome, box camera, IR camera and PTZ, with high definition to 1.3 MP and full frame rate options, networked video recorders (NVR), video central management server (VMS), to big screen display and monitoring solution via network decoders.



The performance of HUS 3.1 is strictly qualified by Honeywell’ standard, aiming to offer our customers the most reliable and the cost effective IP solution. Advances in IP video surveillance are creating boundless business opportunities in the security industry.

Honeywell HUS XPRO 3.1- Pure IP and HD Digital Surveillance System with Powerful Integration
Capability

HUS XPRO 3.1 consists of the following key components:

HUS-VMS - IP Video and Alarm Management Server
HUS-VMS is the core of the entire system. HUS-VMS comes with the pre-installed hardware, software and
bundled Software Support Agreement (SSA) offering customers simple commissioning and system
setup. Each server supports up to 500 channels of video streaming.
Model: HUS-VMS-MAS, HUS-VMS-SLA

HUS-NVR- Networked Video Recorder
  • HUS-NVR is a truly digital and network-based solution.
  • Each NVR supports up to 100 channels of D1 real-time recording and 40 channels of playback
    simultaneously.
  • HUS-NVR standard configuration supports 32TB storage, with optional expansion up to 224TB.

Model: HUS-NVR-5100

HUS-D4 - Network Video Decoder

  • HUS-D4 is the state-of-the-art 4-ch network decoder for multi-screen or video wall monitoring
  • Each HUS-D4 supports 4 big screen monitors at HD up to 1080P independently.
  • Each screen displays up to 2x2 channels at D1 @ 25fps or 4x4 channels at CIF @ 25fps images

Model: HUS-D4

Target Markets
Honeywell HUS XPRO 3.1 supports seamless integration with information management, which is ideal
for medium to large size organizations, as well as for the following specific vertical markets:

  • Public Security (city surveillance)
  • Transportation
  • Banking & Finance
  • Casinos
  • Commercial or Public Buildings
  • Manufacturing & Industries
  • Science or Information Technology Parks

System Diagram


Honeywell HUS XACT 3.1- Cost Effective IP and HD Digital Surveillance System
 
HUS XACT 3.1 consists of the following key components:

HUS XACT- Central Management Server with High Performance Storage Package
HUS XACT is an efficient security platform designed for the medium size IP security solution. It consists
of 3 offerings of different system capacities - HUSXACT-100, HUS-XACT100S and HUS-XACT050S.
The HUS-XACT100 and HUS-XACT100S support up to 100 channels at D1 resolution, full frame
recording, and viewing visit from 5 clients offered by a single server and storage package. The HUS-
XACT050S supports up to 50 channels at D1 resolution and full frame recording. Model: HUS-XACT100,
HUS-XACT100S, HUS-XACT050S

HUS-VSA- Video Storage Extension Array
HUS XACT adopts the advanced HUS-VSA. Its storage box guarantees 24x7 recording with no downtime.
The HUS-VSA consists of 16 HDD bays that support up to 32TB storage capacity. Besides, this model
can be used with HUS-XACT100 only. Model: HUS-VSA-316

Target Markets
Honeywell HUS XACT 3.1 is ideal for medium size IP security systems with less than 100 video channels.
It is a cost effective surveillance and intrusion application, which offers customers a simple deployment at
minimal investment costs. Moreover, the pre-installed software and packages with all-round features well
fulfill market demands of the following sectors:

  • Medium Scale Commercial Buildings
  • Hospital
  • Education
  • Branch Offices
  • Warehouse
  • Shopping Centre or Chain Stores
